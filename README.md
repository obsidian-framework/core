> Core package for the Obsidian Framework - Published on GitHub Packages

**Full documentation**: [obsidian-java.com](https://obsidian-java.com/)

## Installation

### Maven

Add Jitpack Packages repository

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

Add dependency

```xml
<dependency>
    <groupId>com.github.obsidian-framework</groupId>
    <artifactId>core</artifactId>
    <version>v1.2.0</version>
</dependency>
```

## Build & Deploy

```bash
# Build
mvn clean package

# Deploy to GitHub Packages
mvn deploy
```
