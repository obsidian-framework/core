## Installation

### Maven

Add Jitpack Packages repository

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

Add dependency

```xml
<dependency>
    <groupId>com.github.obsidian-framework</groupId>
    <artifactId>core</artifactId>
    <version>latest</version>
</dependency>
```

## Build & Deploy

```bash
# Build
mvn clean package

# Deploy to GitHub Packages
mvn deploy
```
